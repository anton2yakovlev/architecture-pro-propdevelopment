# Требования к внешним интеграциям

## Безопасность

- TLS 1.2+ для всех соединений с партнёрской системой
- Шифрование биометрических данных при передаче
- Соответствие ФЗ-152 при обработке персональных данных
- Rate Limiting для API (100 запросов/минуту)
- Логирование операций доступа

## Аутентификация и авторизация

**Протоколы:**
- OAuth 2.0 + JWT токены (время жизни 15 минут)

**Авторизация:**
- RBAC для управления доступом к устройствам
- Проверка прав собственника на конкретное устройство

## Взаимодействие между системами

**REST API** - управление устройствами (добавление/удаление доступа, список устройств)

**WebSocket** - события в реальном времени (звонки в домофон, открытие шлагбаума)

**Формат данных:** JSON
